### GROUP 1: FRAMING -- WOOD (7 families)

#### Dimension Lumber-Section
- **Category Group**: Framing
- **Orientation**: Section -- cut perpendicular to grain, shows rectangular cross-section
- **Typical Use Cases**: Wall studs, top/bottom plates, headers, rafters, joists, blocking in wood-frame walls and roof assemblies
- **Placement Method**: Point placement
- **Common Scales**: 1" = 1'-0", 1-1/2" = 1'-0", 3" = 1'-0"
- **Paired With**: Plywood-Section, Gypsum Wallboard-Section, Nominal Cut Lumber-Section, Common Wood Nails-Side
- **Line Weight**: 4 (medium -- structural member)
- **Notes**: Most-used wood framing component. Parametric for 2x4, 2x6, 2x8, 2x10, 2x12.

#### 06-FRAMING-WOOD-Section-Stretch1
- **Category Group**: Framing
- **Orientation**: Section -- stretchable wood member
- **Typical Use Cases**: Blocking above doors, cripple studs, misc wood framing that needs non-standard lengths
- **Placement Method**: Line-based (stretchable)
- **Common Scales**: 1" = 1'-0", 1-1/2" = 1'-0", 3" = 1'-0"
- **Paired With**: C Studs-Section, Gypsum Wallboard-Section
- **Line Weight**: 4 (medium)
- **Notes**: Use when standard Dimension Lumber sizes don't fit. Stretches to any length.

#### Nominal Cut Lumber-Section

## REVIT API PLACEMENT NOTES

### Loading Detail Item Families
```
Family.LoadFamily(doc, familyPath)
```
Detail item families are in the `Detail Items` category (BuiltInCategory.OST_DetailComponents).

### Placing a Detail Component
```csharp
// Point-based placement
doc.Create.NewFamilyInstance(
    point,           // XYZ location
    familySymbol,    // The specific type within the family
    detailView       // The drafting/detail view
);
```

### Key Behaviors
- Detail items exist ONLY in the view they are placed in (view-specific)
- They do NOT appear in 3D views or other sections
- Rotation is controlled by the `Rotation` parameter after placement
- For line-based families, use two points to define start/end
- Layer ordering matters: place back-to-front (exterior to interior) for proper overlap

### Scale Considerations
- At 1/4" = 1'-0", only show major components (studs, GWB, sheathing). Skip fasteners and sealants.
- At 1" = 1'-0", show all standard components. This is the default detail scale.
- At 1-1/2" = 1'-0" and 3" = 1'-0", show fasteners, sealant beads, and fine hardware.
