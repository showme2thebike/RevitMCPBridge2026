## Notes for AI Detail Drawing

### Drawing Priority (most commonly needed)
1. **HM door head/jamb** - every commercial project
2. **Wood door head/jamb** - every residential project
3. **Storefront door sill** - every commercial entrance
4. **Threshold details** - every project (ADA compliance)
5. **Elevator entrance** - any multi-story project
6. **Overhead door** - industrial, commercial, residential garage

### ADA Critical Dimensions
- Door clear opening: 32" min (34" nominal door in 36" frame)
- Threshold height: 1/2" max exterior, 1/4" max interior (without bevel)
