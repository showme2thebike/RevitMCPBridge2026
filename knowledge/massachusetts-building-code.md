# Massachusetts Building Code - Key Amendments

**Based on**: 9th Edition Massachusetts Building Code (780 CMR), based on IBC 2015
**Base Code**: IBC 2015 with extensive Massachusetts amendments
**Authority**: Massachusetts Board of Building Regulations and Standards (BBRS)

---

## Quick Reference - Massachusetts vs. IBC

| Requirement | Massachusetts | IBC 2021 |
|-------------|--------------|----------|
| Code edition | IBC 2015 base (9th Ed) | IBC 2021 |
| Energy code | Stretch Code available / base IECC 2021 | IECC 2021 |
| Residential sprinklers | Not statewide required; Stretch Code option | Not required |
| Seismic | SDC B statewide | Per ASCE 7 |
| Accessibility | MAAB (Massachusetts Architectural Access Board) — exceeds ADA | ADA 2010 |
| Snow | Significant — 40-60 psf ground snow load | Per ASCE 7 |

## Energy Code

**Adopted**: Base Energy Code (IECC 2021 base) + optional Stretch Code
**Authority**: Massachusetts BBRS / Board of Building Regulations and Standards

### Climate Zone
- **5A**: Entire state

### Base Code Residential Envelope (IECC 2021 Zone 5A)

| Element | Requirement |
|---------|-------------|
| Ceiling | R-60 (unvented attic R-38+R-10ci) |
| Wall | R-20 or R-13+5ci |
| Floor | R-30 |
| Windows | U-0.30 |
| Doors | U-0.20 |
| Air sealing | 3.0 ACH50 |

### Stretch Code (Adopted by 300+ municipalities including Boston)
- More stringent than base — includes:
  - All-electric ready construction
  - Enhanced air sealing (3.0 ACH50 or better)
  - EV charging infrastructure
  - Solar-ready construction
  - Heat pump water heaters preferred

### Municipal Opt-In
- Boston, Cambridge, Brookline, Newton, Arlington, Lexington, and 300+ municipalities have adopted Stretch Code
- **Verify local adoption before designing mechanical systems**

## Residential Code

**Adopted**: 780 CMR (9th Edition) for residential (based on IRC 2015)
**Authority**: Massachusetts BBRS

### Key Amendments
- **Sprinklers**: Not required statewide for 1-2 family; required under Stretch Code in some municipalities
- **Snow loads**: 40-60 psf ground snow (Boston: 40 psf; Worcester Hills: 55 psf; Berkshires: 60+ psf)
- **Ice dam protection**: Required — 24" from eave minimum ice and water shield
- **Frost depth**: 48" statewide
- **Radon**: Zone 2 (western MA Zone 1) — passive radon system recommended
- **Coastal**: Chapter 91 Waterways Act for tidal/coastal projects

---

## Massachusetts Architectural Access Board (MAAB)

Massachusetts has its own accessibility standard that **exceeds ADA** in many areas:

| Element | MAAB | ADA |
|---------|------|-----|
| Restroom turning radius | 60" diameter | 60" diameter |
| Reach range | 15"-48" | 15"-48" |
| Accessible route width | 44" | 36" |
| Ramp handrail extensions | Required both ends | Required |
| Existing buildings | Stricter trigger thresholds | Less strict |

- **MAAB registration**: All new construction and renovation projects must file with MAAB
- **Variance process**: Available but time-consuming — design to comply upfront
- **Elevator**: Required in most 3+ story buildings (stricter than ADA)

## Seismic / Wind / Natural Hazards

### Seismic
- **Statewide**: SDC B — moderate seismic risk
- Notable historical earthquakes in Cape Ann area (1755 — M6.0)
- Standard IBC seismic provisions for SDC B apply
- **Boston area**: Some liquefaction risk in filled land (Back Bay, South Boston waterfront)

### Wind / Coastal
| Region | Design Wind Speed | Notes |
|--------|-----------------|-------|
| Outer Cape Cod | 140-150 mph | Extreme coastal exposure |
| South Shore / Plymouth | 120-130 mph | |
| North Shore | 115-120 mph | |
| Boston | 110-115 mph | |
| Inland | 100-110 mph | |

- **Coastal flood risk**: Significant — nor'easters and hurricane remnants
- **FEMA flood zones**: Extensive coastal V/AE zones
- **Chapter 91 (Waterways Act)**: Regulates construction within 100' of tidal/great ponds

### Snow
| Region | Ground Snow Load |
|--------|----------------|
| Cape Cod / Islands | 30-35 psf |
| Boston Metro | 40 psf |
| Worcester / Central MA | 45-55 psf |
| Berkshires / Western MA | 55-70 psf |

- **Ice dam prevention**: Critical — 24" minimum ice and water shield at eaves
- **Roof design**: Snow drift loads critical at parapet walls, lower roofs

## Notable State Amendments

- **MAAB filing**: Mandatory for all projects — not optional
- **Stretch Energy Code**: Adopted in most progressive municipalities — design for all-electric
- **Wetlands Protection Act**: 310 CMR 10 — 100-foot buffer from wetlands
- **Historical Commission**: Major presence in Boston, Beacon Hill, historic districts statewide
- **Boston ISD**: City of Boston Inspectional Services has additional local amendments
- **Cambridge**: Has own amendments including net-zero requirements
- **MassDEP**: Environmental review often required — hazmat, UST, wetlands
