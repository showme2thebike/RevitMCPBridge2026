# Washington State Building Code - Key Amendments

**Based on**: Washington State Building Code (WSBC), based on IBC 2021
**Base Code**: IBC 2021 with Washington State amendments
**Authority**: Washington State Building Code Council (SBCC)

---

## Quick Reference - Washington State vs. IBC

| Requirement | Washington State | IBC 2021 |
|-------------|-----------------|----------|
| Energy code | WSEC (IECC 2021 + WA amendments) | IECC 2021 |
| Residential sprinklers | Not statewide required | Not required |
| Seismic | SDC C-D (west of Cascades) | Per ASCE 7 |
| Wildfire WUI | State WUI code (Eastern WA) | Not addressed |
| Radon | Zone 2 (Eastern WA) | Not addressed |

## Energy Code

**Adopted**: Washington State Energy Code (WSEC), one of most stringent in US
**Climate Zones**: 4C (west of Cascades), 5B/6B (east of Cascades)

### Climate Zones

| Zone | Region | Examples |
|------|--------|---------|
| 4C (Marine) | West of Cascades | Seattle, Tacoma, Olympia, Bellingham |
| 5B | Eastern WA lowlands | Spokane, Tri-Cities, Wenatchee |
| 6B | Eastern WA highlands | Colville, Republic |

### Residential Envelope Requirements (WSEC 2021)

| Element | Zone 4C | Zone 5B | Zone 6B |
|---------|---------|---------|---------|
| Ceiling | R-49 | R-49 | R-60 |
| Wall | R-21 | R-21 | R-21+5ci |
| Floor | R-30 | R-30 | R-38 |
| Windows | U-0.28 | U-0.28 | U-0.25 |
| Doors | U-0.20 | U-0.20 | U-0.20 |

### Washington Energy Code Highlights
- **EV charging**: Required for new residential (one EVCS per unit min)
- **Heat pumps**: Preferred over gas for new residential per state policy direction
- **Air sealing**: 3.0 ACH50 max for new residential
- **Lighting**: High-efficacy required in most spaces

## Residential Code

**Adopted**: Washington State Residential Code (WSRC), based on IRC 2021
**Authority**: Washington State Building Code Council

### Key Amendments
- **Sprinklers**: Not required statewide for 1-2 family; many cities (Seattle, Bellevue) require
- **Radon**: Passive radon system required in Zone 2 areas (eastern WA)
- **Seismic**: Enhanced hold-down requirements west of Cascades
- **Energy**: WSEC requirements more stringent than base IRC
- **Wildfire**: State Wildland-Urban Interface Code applies in designated areas

## Seismic / Wind / Natural Hazards

### Seismic
| Region | Typical SDC | Notes |
|--------|-------------|-------|
| Seattle / Puget Sound | C-D | Seattle Fault, Cascadia Subduction Zone |
| Tacoma / Olympia | C-D | Tacoma Fault zone |
| Cascades | B-C | Volcanic hazard |
| Eastern Washington | A-B | Lower risk |
| Coast | D-E | Tsunami + Cascadia risk |

- **Cascadia Subduction Zone**: M8.0-9.0 event; coastal communities require tsunami design
- **Seattle Fault**: Runs through south Seattle — significant local amplification
- **Liquefaction**: Significant risk in Duwamish valley, lake fill areas
- **Tsunami inundation zones**: Coastal and Puget Sound communities

### Wind
- Western WA: 85-90 mph (marine exposure)
- Eastern WA: 90-100 mph (open terrain)
- Olympic Peninsula coast: 100-110 mph

### Wildfire (Eastern WA)
- **State WUI Code**: Applies in designated wildland-urban interface
- Major fire risk in Okanogan, Chelan, Ferry, Stevens counties
- **Ignition-resistant**: Class A roofing, ember-resistant vents required in WUI

### Flooding / Landslides
- FEMA flood zones per local FIRMs
- Landslide/steep slope regulations (Seattle Critical Areas Ordinance, etc.)
- Coastal erosion on outer coast

## Notable State Amendments

- **Climate commitments**: Washington Clean Buildings Act affects commercial buildings
- **All-electric trend**: Seattle and other cities moving toward all-electric new construction
- **Seismic resilience**: Post-2001 Nisqually earthquake enhanced requirements
- **Local amendments significant**: Seattle has substantial local amendments — verify SDCI requirements
- **Puget Sound shoreline**: Shoreline Management Act applies within 200' of water

## Quick Checklist - New Residential

- [ ] Climate zone determined (4C/5B/6B)
- [ ] WSEC compliance — EV charging, air sealing, U-values
- [ ] Seismic design category determined
- [ ] Site in tsunami inundation zone? (coastal/Puget Sound)
- [ ] Site in WUI zone? (Eastern WA)
- [ ] Local sprinkler requirement? (Seattle, Bellevue — check AHJ)
- [ ] Radon passive system? (Eastern WA Zone 2)
- [ ] Shoreline Management Act? (within 200' of water)
- [ ] Landslide/steep slope critical area?
