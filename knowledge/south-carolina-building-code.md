## Residential Code

**Adopted**: South Carolina Residential Code, based on IRC 2018

### Key Amendments
- **Sprinklers**: Not required for 1-2 family statewide
- **Seismic**: Charleston area — SDC C-D; enhanced requirements
- **Coastal**: Wind-borne debris region coastal counties; impact glazing or shutters
- **Termite**: Required statewide — high pressure coastal areas
- **Crawl space**: Encapsulation recommended; vapor barrier required

## Seismic / Wind / Natural Hazards

### Seismic (Charleston is Critical)
| Region | SDC | Notes |
|--------|-----|-------|
| Charleston metro | C-D | 1886 earthquake M7.3 — deadliest SE US |
| Coastal plain | B-C | |
| Upstate (Greenville/Spartanburg) | A-B | |

- **1886 Charleston Earthquake (M7.3)**: Largest known earthquake in SE US — extensive unreinforced masonry vulnerability
- **Charleston**: SDC C-D — significant design requirements; liquefaction risk
- Special inspections required SDC C+

### Wind / Hurricane
| Region | Design Wind Speed | Notes |
|--------|-----------------|-------|
| Outer coast / Hilton Head | 130-140 mph | Wind-borne debris region |
| Charleston coast | 130-135 mph | |
| Columbia / Midlands | 100-105 mph | |
| Upstate | 100 mph | |

### Flooding
- Coastal: V/AE zones — storm surge significant
- Lowcountry: Extensive tidal flooding
- Post-Matthew (2016) / Dorian (2019): Enhanced flood awareness

## Notable State Amendments
- **Charleston seismic + hurricane**: Dual hazard coastal city — rare combination requiring careful design
- **Myrtle Beach**: High tourism construction volume; resort-specific considerations
- **OCRM (Office of Coastal Resource Management)**: Coastal zone permit required within critical areas
