# Illinois Building Code - Key Amendments

**Based on**: Illinois Building Code (IBC 2018 base) — for state-regulated occupancies
**Base Code**: IBC 2018; local adoption varies significantly
**Authority**: Illinois Capital Development Board (state buildings); local jurisdictions for most projects
**Note**: Chicago has its own code — see chicago-building-code.md for Chicago projects

**Adopted**: IECC 2018 — most municipalities

| Element | Zone 4A | Zone 5A |

| Ceiling | R-49 | R-49 |
| Wall | R-13+5ci | R-20 |
| Floor | R-19 | R-30 |

## Residential Code

**Adopted**: IRC 2018 — locally adopted; verify with AHJ

### Key Amendments
- **Sprinklers**: Not required statewide; some municipalities adopt locally
- **Radon**: Zone 1 (northern/central IL — Chicagoland to Peoria area); Zone 2 (south) — passive system recommended statewide
- **New Madrid Seismic Zone**: Far southern Illinois (Cairo area) — SDC B-C; enhanced requirements
- **Tornado**: High risk statewide — no statewide enhanced construction requirement
- **Frost depth**: 42" Chicago area; 30-36" southern Illinois

---

## Seismic / Wind / Natural Hazards

### Seismic
| Region | SDC | Notes |
|--------|-----|-------|
| Chicagoland / Northern IL | A | Very low risk |
| Central Illinois | A-B | Low risk |
| Southern IL (Cairo/Carbondale) | B-C | New Madrid Seismic Zone |

- **New Madrid Zone**: Southern tip of IL — M7+ scenario possible; unreinforced masonry at risk

### Wind / Tornado
- **Tornado risk**: Significant statewide — averages 50+ tornadoes/year
- **April 1965 Palm Sunday outbreak**: 271 tornadoes in Midwest; IL heavily impacted
- **Design wind**: 90-100 mph statewide
- No enhanced statewide tornado construction provisions

### Snow
| Region | Ground Snow Load |
|--------|----------------|
| Chicago metro | 25-30 psf |
| Rockford / Galena | 25-35 psf (lake-effect from Lake Michigan possible) |
| Peoria / Springfield | 20-25 psf |
| Southern IL | 15-20 psf |

### Radon
- Zone 1: Northern and central Illinois — significant risk
- Chicago area has elevated levels despite urban environment
- Passive system recommended statewide; active in Zone 1

## Notable State Amendments
- **Chicago code is separate**: All Chicago projects use Chicago Building Code — see chicago-building-code.md
- **Suburban Cook County**: Many municipalities adopt local codes; verify each AHJ
- **Rockford, Peoria, Springfield**: Each municipality adopts own code — call building department
- **Illinois Commerce Commission**: Regulates some utility/infrastructure
- **IDOT**: IL Dept of Transportation involvement for projects near state roads
