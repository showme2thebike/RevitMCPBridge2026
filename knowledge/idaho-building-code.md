# Idaho Building Code - Key Amendments

**Based on**: Idaho Building Code Act, based on IBC 2018
**Base Code**: IBC 2018 with Idaho amendments
**Authority**: Idaho Division of Building Safety (DBS)

---

## Quick Reference - Idaho vs. IBC

| Requirement | Idaho | IBC 2021 |
|-------------|-------|----------|
| Code edition | IBC 2018 | IBC 2021 |
| Energy code | IECC 2018 | IECC 2021 |
| Residential sprinklers | Not required statewide | Not required |
| Seismic | SDC B-C | Per ASCE 7 |
| Radon | Zone 1-2 | Not addressed |
| Wildfire | WUI provisions | Not addressed |

## Residential Code

**Adopted**: Idaho Residential Building Code, based on IRC 2018

### Key Amendments
- **Sprinklers**: Not required statewide
- **Radon**: Zone 1 (northern/central Idaho) — passive system strongly recommended
- **Wildfire**: WUI provisions in forested areas — significant risk
- **Frost depth**: 24" southern Idaho; 36-42" northern Idaho
- **Snow**: 25-60 psf depending on elevation and location

---

## Seismic / Wind / Natural Hazards

### Seismic
| Region | SDC | Notes |
|--------|-----|-------|
| Boise / Treasure Valley | B-C | Some fault zones |
| Northern Idaho | B | Lower risk |
| Eastern Idaho | B-C | Wasatch-Yellowstone zone edge |

### Wildfire
- Significant WUI risk throughout forested areas
- Northern Idaho (panhandle): High forest fire risk
- Class A roofing, ember-resistant vents in WUI zones

### Snow
- Boise: 15-20 psf
- Mountain communities: 60-100+ psf (structural engineer required)

## Quick Checklist - New Residential
- [ ] Climate zone (5B southern / 6B northern)
- [ ] Radon passive system (Zone 1 northern/central ID)
- [ ] WUI zone? Wildfire construction requirements
- [ ] Snow load (mountain communities — structural engineer)
- [ ] Frost depth (24-42" by location)
