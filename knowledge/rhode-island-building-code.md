# Rhode Island Building Code - Key Amendments

**Based on**: Rhode Island State Building Code, based on IBC 2018
**Base Code**: IBC 2018 with Rhode Island amendments
**Authority**: Rhode Island State Building Code Standards Committee, Department of Business Regulation

## Energy Code

**Adopted**: IECC 2021 (Rhode Island is among the earlier adopters of the current edition)
**Climate Zone**: 5A statewide

| Element | Zone 5A |
|---------|---------|
| Ceiling | R-49 |
| Wall | R-20 or R-13+5ci |
| Floor | R-30 |
| Windows | U-0.30 |
| Air sealing | 3.0 ACH50 |

### Rhode Island Energy Notes
- **Energy efficiency**: Small state; statewide utility (National Grid) incentive programs active
- Above-code performance increasingly common in Providence metro

## Residential Code

**Adopted**: Rhode Island State Building Code Residential Section, based on IRC 2018

### Key Amendments
- **Sprinklers**: Not required statewide for 1-2 family; some municipalities adopt locally
- **Coastal**: Atlantic coast + Narragansett Bay — FEMA VE/AE zones; post-Sandy updated maps; NFIP compliance critical
- **Radon**: Zone 1 (northwestern RI — granite/crystalline rock) to Zone 2; passive system recommended statewide
- **Frost depth**: 48" statewide
- **Wind-borne debris**: Shore communities (South Kingstown, Narragansett, Westerly) — impact glazing or shutters

## Seismic / Wind / Natural Hazards

### Seismic
- SDC B statewide — moderate risk
- Standard provisions apply

### Hurricane / Coastal
| Region | Design Wind Speed | Notes |
|--------|-----------------|-------|
| Block Island / Atlantic coast | 115-120 mph | Direct Atlantic exposure |
| Narragansett Bay coast | 110-115 mph | Bay amplifies surge |
| Providence / inland | 100-110 mph | |

- **1938 Great New England Hurricane**: Category 3 — Providence storm surge flooded downtown 13 feet; 317 killed in RI alone; remains the defining coastal catastrophe for the state
- **Bob (1991)**: Significant damage to coastal RI
- **Sandy (2012)**: Significant coastal flooding; FEMA maps updated
- **Narragansett Bay**: Storm surge funnels into bay and hits Providence at apex — no barrier island protection

### Snow
| Region | Ground Snow Load |
|--------|----------------|
| Providence / coastal | 30-35 psf |
| Worcester County border (northwest) | 35-40 psf |

- **Blizzards**: Nor'easters regularly produce 24"+ storms; 1978 Blizzard of '78 still referenced
- Ice storm loading important for roof design

## Notable State Amendments
- **Providence**: Largest city; strong local enforcement; historic district review significant (lots of pre-1900 buildings)
- **Newport**: Historic resort; strict design review; coastal FEMA compliance critical
- **Westerly / Narragansett**: Shore community enforcement for coastal construction
- **DEM**: RI Dept of Environmental Management — coastal resources management permits (CRMC)
- **CRMC**: Coastal Resources Management Council — distinct from building permit; required for coastal zone work

## Quick Checklist - New Residential
- [ ] Climate zone: 5A statewide
- [ ] Coastal? FEMA FIRM (post-Sandy); CRMC permit required
- [ ] Narragansett Bay shore? Storm surge — 1938 Hurricane precedent for 13-foot surge
- [ ] Radon passive system (Zone 1-2 — recommended)
- [ ] Frost depth: 48" statewide
- [ ] Wind-borne debris region (shore communities)
