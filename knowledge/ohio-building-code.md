# Ohio Building Code - Key Amendments

**Based on**: Ohio Building Code (OBC), based on IBC 2017
**Base Code**: IBC 2017 with Ohio amendments
**Authority**: Ohio Board of Building Standards (BBS), Ohio Department of Commerce

---

| Requirement | Ohio | IBC 2021 |

## Residential Code

**Adopted**: Ohio Residential Code, based on IRC 2017
**Authority**: Ohio BBS

### Key Amendments
- **Sprinklers**: Not required for 1-2 family
- **Radon**: Zone 1-2 — passive radon system recommended; some local jurisdictions require
- **Snow**: Lake-effect snow significant near Lake Erie — Cleveland/Chardon 60-100+ inches annually
- **Frost depth**: 36" northern Ohio; 30" southern Ohio
- **Carbon monoxide**: CO detectors required per Ohio code

---

## Seismic / Wind / Natural Hazards

### Seismic
- **Statewide**: SDC A-B — low seismic risk
- **Anna Seismic Zone**: Anna, Ohio has experienced M5+ historical earthquakes — local awareness
- Standard IBC provisions apply; no special requirements

### Wind
- **Statewide**: 90-100 mph design wind speeds
- **Lake Erie exposure**: Enhanced wind at coastal properties
- Tornado risk: Moderate — Ohio averages 15-20 tornadoes annually
- No enhanced statewide tornado provisions

### Snow / Lake Effect
| Region | Ground Snow Load | Notes |
|--------|----------------|-------|
| Cleveland / Chardon | 30-40 psf | Lake-effect belt |
| Lake Erie coastal | 35-45 psf | |
| Columbus / Dayton | 20-25 psf | |
| Cincinnati | 15-20 psf | |

- **Lake-effect snow belt**: Chardon, Ashtabula, Geauga County — extreme localized snow
- **Snow drift**: Critical at parapet walls and roof level changes
- **Ice damming**: Ice and water shield required at eaves

### Flooding
- FEMA flood zones along Ohio River (southern border)
- Muskingum, Scioto, Cuyahoga river floodplains
- Great Lakes coastal flooding

## Notable State Amendments

- **OBC Chapter 1 — Administration**: Ohio has own administrative provisions for plan review, inspections
- **Ohio Fire Code**: Follows IFC 2017; significant for suppression system design
- **Elevator**: Ohio has own elevator code and inspection program
- **Radon**: Zone 1 (northwest Ohio) and Zone 2 (most of state) — include passive system
- **Columbus, Cleveland**: Have local amendments; verify with city building departments

## Quick Checklist - New Residential

- [ ] Climate zone (4A southern / 5A northern)
- [ ] OECC energy compliance
- [ ] Radon passive system (statewide recommendation; Zone 1 NW Ohio)
- [ ] Snow load — lake-effect zone? Structural engineer for heavy snow
- [ ] Frost depth (30" south / 36" north)
- [ ] CO detector specified
- [ ] FEMA flood zone (Ohio River / major rivers)
- [ ] Local amendments — Columbus, Cleveland, Cincinnati
