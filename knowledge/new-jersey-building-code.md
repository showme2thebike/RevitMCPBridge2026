# New Jersey Building Code - Key Amendments

**Based on**: New Jersey Uniform Construction Code (NJ UCC), based on IBC 2015
**Base Code**: IBC 2015 with New Jersey amendments
**Authority**: New Jersey Department of Community Affairs (DCA), Division of Codes and Standards

---

## Quick Reference - New Jersey vs. IBC

| Requirement | New Jersey | IBC 2021 |
|-------------|-----------|----------|
| Code edition | IBC 2015 | IBC 2021 |
| Energy code | IECC 2015 | IECC 2021 |
| Residential sprinklers | **Required** — new 1-2 family | Not required |
| Seismic | SDC B | Per ASCE 7 |
| Hurricane / coastal | Significant coast exposure | Per ASCE 7 |
| Radon | Zone 2 | Not addressed |

## Residential Code — SPRINKLER REQUIREMENT

**New Jersey requires fire sprinklers in ALL new one- and two-family dwellings** (since 2018).

- **System type**: NFPA 13D
- **Statewide mandate**: No local opt-out
- **Design**: Full coverage required

### Other Key Amendments
- **Radon**: Zone 2 statewide — passive radon system required for new residential; some municipalities require active
- **Coastal**: Post-Sandy (2012) enhanced flood and wind requirements along shore
- **Frost depth**: 36" statewide
- **Snow**: 25-40 psf

## Seismic / Wind / Natural Hazards

### Seismic
- SDC B statewide — moderate risk
- Standard provisions apply

### Wind / Coastal
| Region | Design Wind Speed | Notes |
|--------|-----------------|-------|
| Atlantic City / Jersey Shore | 120-130 mph | |
| Newark / NYC suburbs | 100-110 mph | |
| Inland | 100 mph | |

- **Post-Sandy (2012)**: Major code and FEMA map updates for coastal areas
- **CAFRA (Coastal Area Facility Review Act)**: Permit required for development in coastal zone
- **Wind-borne debris**: Shore communities — impact glazing or shutters

### Snow
- Statewide: 25-40 psf ground snow load
- Higher in northern NJ / Bergen County area

### Flooding
- **Sandy legacy**: Significant FEMA map updates; many areas in AE/VE zones
- **Advisory Base Flood Elevations**: Post-Sandy ABFE often 2-4' above previous BFE
- CAFRA zone: Strict coastal development regulations

---

## Notable State Amendments
- **Sprinklers mandatory**: Non-negotiable — all new residential
- **CAFRA**: Coastal development permit — significant process
- **NJ DEP**: Environmental permits often required alongside building permits
- **Urban Enterprise Zones**: Some permit fee exemptions
- **Edison, Toms River, Jersey City**: Each have local DCA-registered code enforcement offices
