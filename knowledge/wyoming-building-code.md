## Residential Code

**Adopted**: IRC 2018 — locally adopted; verify with each municipality

### Key Amendments
- **Radon**: Zone 1 statewide — active mitigation system recommended
- **Wind**: Extreme wind statewide — design critical
- **Snow**: High throughout state; mountains extreme
- **Frost depth**: 36-48" statewide
- **Seismic**: Yellowstone area — monitor for local requirements

---

## Seismic / Wind / Natural Hazards

### Seismic
| Region | SDC | Notes |
|--------|-----|-------|
| Yellowstone | C-D | Supervolcano caldera — M6+ common |
| Jackson Hole | B-C | Teton fault zone |
| Rest of state | A-B | Low risk |

- **Yellowstone**: High seismicity — Hebgen Lake 1959 (M7.3), frequent M3-4+ swarms

### Wind (Critical)
- **Statewide**: 100-115 mph design wind speeds
- **Laramie / Cheyenne corridor**: Among windiest inhabited areas in US — 100+ mph gusts common
- **Medicine Bow area**: Extreme persistent wind; structural design critical
- Wind-driven snow: Significant — structural and envelope implications

### Snow
| Region | Ground Snow Load |
|--------|----------------|
| Cheyenne / eastern WY | 20-30 psf |
| Casper | 25-35 psf |
| Laramie | 35-50 psf |
| Jackson Hole / Tetons | 80-150 psf |
| Snowy Range | 150-250+ psf |

### Radon
- **Zone 1**: Entire state — highest EPA category
- Active soil depressurization system recommended statewide
