# Connecticut Building Code - Key Amendments

**Based on**: Connecticut State Building Code, based on IBC 2018
**Base Code**: IBC 2018 with Connecticut amendments
**Authority**: Connecticut Department of Administrative Services, Construction Services; local building officials

## Energy Code

**Adopted**: IECC 2021 — CT adopted 2021 edition (among earlier adopters)
**Climate Zone**: 5A statewide

| Element | Zone 5A |
|---------|---------|
| Ceiling | R-49 |
| Wall | R-20 or R-13+5ci |
| Floor | R-30 |
| Windows | U-0.30 |
| Air sealing | 3.0 ACH50 (blower door required) |

### Connecticut Energy Notes
- CT Green Bank programs incentivize above-code performance
- ENERGY STAR certification widely pursued

## Residential Code

**Adopted**: Connecticut One and Two Family Dwelling Code, based on IRC 2018

### Key Amendments
- **Sprinklers**: Not required statewide for 1-2 family; some municipalities adopt locally
- **Coastal**: Post-Sandy enhanced FEMA flood requirements; NFIP compliance critical
- **Radon**: Zone 1 (northwest CT — granite traprock ridges), Zone 2 (remainder); passive system strongly recommended
- **Frost depth**: 48" statewide
- **Wind-borne debris**: Shore communities — verify local requirements for impact glazing

---

## Seismic / Wind / Natural Hazards

### Seismic
- SDC B statewide — moderate risk
- Some minor fault systems; 1982 Moodus, CT area had notable microseismic activity
- Standard provisions apply; special inspections for SDC B triggered at certain sizes

### Hurricane / Coastal
| Region | Design Wind Speed | Notes |
|--------|-----------------|-------|
| Shore / New London / Milford | 110-120 mph | Direct Atlantic coast exposure |
| Coastal CT (Long Island Sound) | 105-115 mph | Protected by Long Island |
| Hartford / inland | 100-105 mph | |

- **1938 Great New England Hurricane**: Category 3 storm surge 10-17' — 100+ killed in CT; defining event for coastal risk
- **Sandy (2012)**: Significant damage along CT coast; Bridgeport, Westport, Milford hard hit
- **Long Island Sound**: Protected from direct ocean swells but storm surge significant
- FEMA flood maps updated post-Sandy; verify current FIRM

### Snow
| Region | Ground Snow Load |
|--------|----------------|
| Hartford / central CT | 30-35 psf |
| New Haven / coast | 25-30 psf |
| Litchfield Hills (northwest) | 35-45 psf |

- **Nor'easters**: Major snow events; ice storms significant — design for ice loads

## Notable State Amendments
- **Historic buildings**: CT has extensive pre-code building stock; adaptive reuse common in cities
- **Bridgeport / New Haven / Hartford**: Urban municipalities with active local code enforcement
- **Greenwich / Westport / Fairfield County**: High-value coastal construction; strict local enforcement and local amendments common
- **CT DEEP**: Environmental permits for coastal/wetland work

## Quick Checklist - New Residential
- [ ] Climate zone: 5A statewide
- [ ] Coastal? FEMA FIRM — post-Sandy flood elevation; NFIP compliance
- [ ] Radon passive system (Zone 1-2 — recommended)
- [ ] Frost depth: 48" statewide
- [ ] Nor'easter snow — roof drift loads at valleys and parapets
- [ ] Shore community? Wind-borne debris — local requirements vary
