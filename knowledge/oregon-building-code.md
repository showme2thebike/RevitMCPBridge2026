# Oregon Building Code - Key Amendments

**Based on**: 2021 Oregon Structural Specialty Code (OSSC)
**Base Code**: IBC 2021 with Oregon amendments
**Authority**: Oregon Building Codes Division (BCD), Department of Consumer & Business Services

---

## Quick Reference - Oregon vs. IBC

| Requirement | Oregon | IBC 2021 |
|-------------|--------|----------|
| Energy code | OEESC (IECC 2021 base) | IECC 2021 |
| Residential sprinklers | Not statewide; many jurisdictions require | Not required |
| Seismic | SDC C-D west of Cascades | Per ASCE 7 |
| Wildfire WUI | Oregon WUI Code applies | Not addressed |
| Radon | Zone 2-3 (low-moderate) | Not addressed |
| Accessibility | Oregon ADA + state amendments | ADA 2010 |

## Energy Code

**Adopted**: Oregon Energy Efficiency Specialty Code (OEESC), based on IECC 2021
**Authority**: Oregon Building Codes Division

### Climate Zones

| Zone | Region | Examples |
|------|--------|---------|
| 4C | Coast/western valleys | Portland, Salem, Eugene |
| 5B | Eastern Oregon | Bend, Medford (inland) |

## Residential Code

**Adopted**: Oregon Residential Specialty Code (ORSC), based on IRC 2021
**Authority**: Oregon Building Codes Division

### Key Amendments
- **Sprinklers**: No statewide mandate; Portland, Bend, and many jurisdictions adopt locally
- **Radon**: Passive radon system recommended in Zone 2 areas; required in some jurisdictions
- **Seismic bracing**: Enhanced requirements west of Cascades — continuous hold-downs common
- **Wildfire**: Oregon WUI Code applies in designated areas (eastern Oregon, southern Oregon)
- **Energy**: Stretch code option available; some local governments require above base

## Seismic / Wind / Natural Hazards

### Seismic
| Region | Typical SDC | Notes |
|--------|-------------|-------|
| Portland/Willamette Valley | C-D | Cascadia Subduction Zone risk |
| Coast | D-E | Highest risk — tsunami zone |
| Eastern Oregon | A-B | Lower seismic risk |

- **Cascadia Subduction Zone**: M8.0-9.0 event probability; coastal areas require tsunami-resilient design
- **Portland Hills Fault**: Active fault through metro area
- **Special inspections**: Required for SDC C+ per IBC Chapter 17

### Wind
- Western Oregon: 85-90 mph design (exposure B/C)
- Columbia River Gorge: Enhanced wind design required (high wind corridor)
- Eastern Oregon: 90-100 mph design

### Wildfire
- **Oregon WUI Code**: Applies in State Forestland-Urban Interface
- **Defensible space**: 100' required in WUI zones
- **Ignition-resistant construction**: Class A roofing, enclosed eaves, ember-resistant vents
- **Eastern/Southern Oregon**: Significant WUI exposure

### Flooding
- FEMA flood zones per local FIRM maps
- Coastal areas: V zones for tsunami/wave action
- Willamette River flood plains

---

## Notable State Amendments

- **Mandatory energy compliance**: OEESC strictly enforced statewide
- **Seismic hold-downs**: Continuous rod systems common for multi-story light frame in SDC C-D
- **Tsunami evacuation signage**: Required in designated coastal inundation zones
- **Oregon WUI Code**: Separate code chapter for wildfire interface construction
- **Local amendments**: Portland, Eugene, Bend frequently adopt more stringent standards
- **Soft story**: Portland has voluntary retrofit program; not yet mandatory
