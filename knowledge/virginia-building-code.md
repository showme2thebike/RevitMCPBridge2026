# Virginia Building Code - Key Amendments

**Based on**: Virginia Uniform Statewide Building Code (USBC), based on IBC 2018
**Base Code**: IBC 2018 with Virginia amendments
**Authority**: Virginia Department of Housing and Community Development (DHCD)

---

## Quick Reference - Virginia vs. IBC

| Requirement | Virginia | IBC 2021 |
|-------------|----------|----------|
| Code edition | IBC 2018 | IBC 2021 |
| Energy code | VECC (IECC 2018 base) | IECC 2021 |
| Residential sprinklers | Not required statewide | Not required |
| Seismic | SDC A-B (most); B-C Northern VA | Per ASCE 7 |
| Hurricane | Coastal — ASCE 7 wind | Per ASCE 7 |
| Accessibility | USBC Chapter 11 (ADA equivalent) | ADA 2010 |

**Authority**: Virginia DHCD

## Residential Code

**Adopted**: USBC Residential provisions, based on IRC 2018
**Authority**: Virginia DHCD — local building departments enforce

### Key Amendments
- **Sprinklers**: Not required for 1-2 family statewide
- **Radon**: Zone 2-3 (western VA Zone 2; eastern VA Zone 3) — passive system recommended
- **Coastal**: Hampton Roads area — enhanced flood and wind design
- **Mountain**: Western VA — snow loads, frost depth 24-30"
- **Frost depth**: 18" eastern VA; 24-30" western mountains

## Seismic / Wind / Natural Hazards

### Seismic
| Region | Typical SDC | Notes |
|--------|-------------|-------|
| Northern VA / DC suburbs | B-C | Some proximity to Eastern Seaboard seismicity |
| Richmond / Central VA | A-B | |
| Western Mountains | A-B | |
| Coastal / Hampton Roads | A-B | |

- **2011 Mineral, VA earthquake (M5.8)**: Largest eastern US quake in decades — affected DC, Richmond
- Standard IBC seismic provisions apply; no special enhanced requirements

### Wind / Hurricane
| Region | Design Wind Speed | Notes |
|--------|-----------------|-------|
| Virginia Beach / Outer Banks | 130-140 mph | Wind-borne debris region |
| Hampton Roads | 120-130 mph | Hurricane risk |
| Northern VA / Richmond | 100-110 mph | |
| Mountains | 100 mph | |

- **Hurricane risk**: Hampton Roads, Virginia Beach significant exposure
- **Nor'easters**: Coastal flooding and wind damage
- **Wind-borne debris**: Within 1 mile of coast where V ≥ 130 mph

### Snow
| Region | Ground Snow Load |
|--------|----------------|
| Virginia Beach / Coast | 15-20 psf |
| Richmond / Northern VA | 25-30 psf |
| Blue Ridge / Shenandoah | 30-40 psf |
| Alleghany Highlands | 40-60 psf |

### Flooding
- **Hampton Roads**: One of highest sea level rise rates in US — 3-5mm/yr subsidence plus climate
- **FEMA flood zones**: Extensive AE/VE zones coastal
- **Tidal flooding**: Regular king tide flooding in Norfolk, Hampton Roads
- **Chesapeake Bay**: Shoreline erosion and flooding

## Notable State Amendments

- **Hampton Roads sea level rise**: Chronic flooding increasingly drives building elevation requirements
- **Northern VA local amendments**: Arlington, Alexandria, Fairfax have local amendments; verify
- **BOCA**: Virginia was historically a BOCA code state; some institutional memory in plan review departments
- **Manufactured housing**: DHCD regulates manufactured housing separately
- **Elevator**: Virginia has own elevator code and inspection program (DPOR)

## Quick Checklist - New Residential

- [ ] Climate zone (3A coast / 4A central / 5A mountains)
- [ ] VECC energy compliance (IECC 2018 base)
- [ ] Coastal? Wind speed, flood zone, impact glazing
- [ ] Hampton Roads? Sea level rise / chronic flooding — elevate above BFE
- [ ] Snow load (mountains — verify structural)
- [ ] Frost depth (18" east / 24-30" mountains)
- [ ] Radon passive system (Zone 2-3)
- [ ] Local amendments (Northern VA — Arlington, Fairfax, Alexandria)
