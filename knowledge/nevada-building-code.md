# Nevada Building Code - Key Amendments

**Based on**: Nevada Revised Statutes Chapter 461A; local adoption varies
**Base Code**: IBC 2018 (most jurisdictions); Clark County/Las Vegas on own adoption cycle
**Authority**: Nevada State Public Works Division for state buildings; local jurisdictions for private

## Energy Code

**Adopted**: IECC 2018 — locally adopted; Clark County may vary
**Climate Zones**: 3B (Las Vegas / southern NV), 5B (Reno / northern NV)

| Element | Zone 3B | Zone 5B |
|---------|---------|---------|
| Ceiling | R-38 | R-49 |
| Wall | R-13 | R-20 |
| Floor | R-13 | R-30 |
| Windows | U-0.40, SHGC 0.25 | U-0.30 |

### Nevada Energy Notes
- **Las Vegas**: 115°F+ summer peaks; cooling load dominant
- **SHGC**: Critical in southern Nevada — limit solar heat gain
- **Radiant barrier**: Highly beneficial in attics — 10-15°F attic temp reduction

## Residential Code

**Adopted**: IRC 2018 — locally adopted

### Key Amendments
- **Sprinklers**: Not required statewide; Clark County/Henderson may have specific requirements — verify
- **Extreme heat**: HVAC sizing critical — Manual J required; larger systems common
- **Wildfire**: Northern Nevada (Reno/Sparks area, Carson City) — WUI provisions
- **Frost depth**: Minimal Las Vegas (0-6"); Reno (18-24"); rural northern NV (24-36")
- **Seismic**: Reno area — enhanced requirements

## Seismic / Wind / Natural Hazards

### Seismic
| Region | SDC | Notes |
|--------|-----|-------|
| Reno / Sparks / Carson City | C | Active fault system — 1954 Dixie Valley M7.1 nearby |
| Las Vegas | A-B | Low risk |
| Rural Nevada | A-B | Basin and Range faulting |

- **Walker Lane**: Active seismic zone through Reno — significant fault proximity
- **Reno**: SDC C design requirements; some liquefaction risk in fluvial deposits

### Wind
- Las Vegas: 90-100 mph
- Reno: 90-100 mph (Washoe Valley — gusts to 100 mph documented)
- No hurricane exposure

### Wildfire
- Northern Nevada: Significant grass and brush fire risk
- Washoe, Douglas, Carson City counties — WUI interface
- Reno 2020 wildfire — Reno suburbs directly threatened

### Extreme Heat (Las Vegas)
- Record high: 117°F (2017)
- **Design temperatures**: 110-115°F outdoor design; 80-85°F wet bulb
- **Concrete**: Early morning pours in summer required
- **Dark surfaces**: Avoid — urban heat island amplification

### Flooding
- Las Vegas: Flash flooding in desert — street flooding during rare heavy rain
- Clark County Flood Control District: Extensive wash/channel system
- Reno: Truckee River flooding; 1997 major flood event

## Notable State Amendments
- **Las Vegas / Clark County**: Clark County Development Services has own building code enforcement; verify local edition
- **Las Vegas building volume**: One of highest per-capita construction volumes in US
- **Gaming/hotel**: Major building type; extensive fire suppression requirements
- **Reno**: Rapidly growing tech hub; verify current Washoe County edition
